# Backend con Node.js y Express Generator  
  
Este proyecto utiliza **Node.js v24.13.1** y **npm** como dependencias principales. Asegúrate de tener instaladas estas versiones o superiores antes de ejecutar el proyecto.  

## Requisitos Previos  
  
- Node.js (El proyecto utiliza v24.13.1)
- npm 
  
## Instalación  
1. Clona el repositorio en tu máquina local
   ```bash  
   git clone https://github.com/RafaellMoralesL/ToDoList-Backend.git```  
  
2. Ingresa al directorio del proyecto  
    ```bash  
    cd backend  
  
3. Instala las dependencias necesarias  
    ```bash  
    npm install  
   
## Ejecución del proyecto  
  
Para ejecutar el proyecto se utiliza el comando  
    ```bash  
    npm start
